Description
Ce projet, réalisé dans le cadre de ma formation à l'INPT, est une application de gestion de cabinet médical développée en Java. Elle vise à dématérialiser l'ensemble du processus de consultation, depuis la prise de rendez-vous jusqu'à la facturation finale.
